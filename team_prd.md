# 견종 추천 서비스 PRD

## 1. 문제 정의

사용자는 직접 키워보기 전까지 자신에게 어떤 견종이 적합한지 판단하기 어렵다.

## 2. 타깃 사용자

강아지를 키우고 싶지만, 어떤 견종을 선택해야 할지 모르는 사람

## 3. 제품 목표

사용자의 성향과 생활 환경을 바탕으로 적합한 견종을 추천한다.

## 4. 핵심 기능

- MBTI, 직업, 주거지 등 사용자 데이터를 수집한다.
- 객관식·주관식 설문 결과와 견종 정보를 바탕으로 AI가 견종을 추천한다.
- 최종적으로 적합도 순위가 포함된 견종 3종을 제공한다.

## 5. 비목표

- 강아지 입양 관련 정보 제공
- 계정 기능

## 6. 성공 기준

AI가 사용자에게 적합하다고 판단한 견종을 1순위, 2순위, 3순위로 추천한다.
