# 견종 추천 서비스 PRD

## 1. 문제 정의

사용자는 직접 키워보기 전까지 자신에게 어떤 견종이 적합한지 판단하기 어렵다.

## 2. 타깃 사용자

강아지를 키우고 싶지만, 어떤 견종을 선택해야 할지 모르는 사람

## 3. 제품 목표

사용자의 성향과 생활 환경을 바탕으로 적합한 견종을 추천한다.

## 4. 핵심 기능

- MBTI, 직업, 주거지 등 사용자 데이터를 수집한다.
- 객관식·주관식 설문 결과와 견종 정보를 바탕으로 AI가 견종을 추천한다.
- 최종적으로 적합도 순위가 포함된 견종 3종을 제공한다.

## 5. 비목표

- 강아지 입양 관련 정보 제공
- 계정 기능

## 6. 성공 기준

AI가 사용자에게 적합하다고 판단한 견종을 1순위, 2순위, 3순위로 추천한다.

## 7. 견종 추천 내부 구현 방식

### 1단계: 후보군 선정

객관식 설문 결과와 견종 정보를 알고리즘에 입력해 사용자에게 가장 적합하다고 판단되는 후보 견종 5종을 추린다.

### 2단계: 에이전틱 AI 기반 최종 추천

에이전틱 AI가 객관식 설문 결과, 주관식 설문 결과, 후보 견종 5종의 정보를 종합한다. 추가 정보가 필요하다고 판단하면 웹 검색 도구를 자율적으로 사용하고, 검색 결과를 추천 판단에 반영해 최종 추천 견종 3종을 선정한다.

## 8. 추천 결과

- 1순위 견종
- 2순위 견종
- 3순위 견종
